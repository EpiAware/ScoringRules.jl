## Nature of this port

The **initial port was generated by a large language model (Claude), under human
guidance**, by translating the R `scoringRules` package (version 1.1.3). Every
closed-form scoring-rule expression and every algorithm in that initial version
is a translation of the original R and C++ implementation by the authors listed
above; the port only adapts the interface to idiomatic Julia (multiple dispatch
on [Distributions.jl](https://github.com/JuliaStats/Distributions.jl) types) and
introduces no new statistical methodology.

This is a **living package and is expected to diverge from upstream over time**.
Future versions may add methods, change interfaces, or make implementation
choices that differ from the R package. That divergence does not change where the
methods came from, so please keep citing the original authors regardless of how
far the package moves on.

Any errors introduced in translation, or in later development, are the
responsibility of this package and its maintainers, not of the original authors.
If you find a discrepancy with the R package, please report it as a bug here.
